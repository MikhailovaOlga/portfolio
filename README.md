## Ключевые проекты

### 1. FUNплан — Агрегатор мероприятий досуга

**Описание:** Мобильное приложение для поиска и планирования культурных, спортивных и развлекательных событий. Собирает данные из сотен источников, фильтрует по интересам, предоставляет прямые ссылки на билеты и аналитику для организаторов.

**Ключевые результаты:**

- 95% охвата городских событий
- Удержание пользователей > 15% на 28-й день
- Roadmap на 3 выпуска с приоритизацией функций

### 2. Планировщик Т-Банк — Умный задачник в банковском приложении

**Описание:** Интеграция планировщика задач в мобильное приложение банка. Пользователи ставят задачи (покупки, поездки, платежи), получают напоминания и переходят в сервисы банка/партнеров с 5% кэшбэком за выполнение.

**Ключевые результаты:**

- Рост транзакций по картам +33% (с 12 до 16/мес)
- 30% активных пользователей банка используют планировщик
- Интеграция с сервисами банка и партнерами
